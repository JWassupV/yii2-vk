Установка
------------
Предпочтительный способ установить это расширение через [composer](http://getcomposer.org/download/).

Либо запустить

```
php composer.phar require --prefer-dist jumper423/yii2-vk "*"
```

или добавить

```
"jumper423/yii2-vk": "*"
```

в файл `composer.json`.
